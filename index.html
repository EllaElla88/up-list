傳統在 HTML 上排板都是採用表格（table）的方式，但是用表格排板會衍伸出許多的缺點，例如表格的架構必須等整個 table 完整下載後，才能順利呈現，如果表格內的東西相當多（例如圖檔或影片檔）的時候，就可以感覺到網頁開啟速

度較慢，另外是 Table的標籤過多，一方面整理起來比較花時間，另一方面是搜尋引擎比較不容易看懂你的程式碼。現在新一代的設計採用 CSS + DIV 的排板方式，取代傳統的 table 排板，並解決舊有的問題，提升網頁開起速度，這篇就是要告訴你如何使用 DIV 標籤。DIV 可以解釋為區塊，用 DIV 標籤包起來的東西，瀏覽器會視為一個物件，大致長成這樣 內容，你可以用 div 將網頁內容的各區塊包起來，再去做 CSS 排板，目前所有的主流瀏覽器都支援 DIV 標籤。
